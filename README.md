# Precious Omole — Portfolio

This is the personal portfolio site for Precious Omole, SDR and Customer Experience Professional.

## How to use on GitHub Pages

1. Create a new GitHub repository named: `yourusername.github.io`
2. Upload both files in this ZIP (index.html and README.md)
3. Go to Settings → Pages → set source to main branch
4. Your site will be live at: `https://yourusername.github.io`

## To update the site

Just click the pencil icon on index.html in GitHub and edit directly in your browser.
